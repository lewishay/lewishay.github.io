BlueLime (2025)

BlueLime is a music player application for Windows.

Features include:

  • Playback of local audio files, including changing playback speed and pitch
  • Responsive UI that adapts to window size
  • Loading a music library from a specified directory, automatically saving library data and album art to a cache for quicker subsequent loading times
  • Asynchronous scanning of file metadata to obtain track information and album art, updating the cache when new files have been added or when files have been updated/deleted
  • Settings to assign custom keybinds, change the library directory, reload library and rebuild caches

Technologies used:

  • C# and XAML
  • Windows Presentation Foundation (WPF)
  • Visual Studio
  • .NET packages: NAudio, SoundTouch.Net, TagLibSharp, Newtonsoft.Json
  • MSTest for unit and integration testing

Ultex (2023-2025)

Ultex is a 2D space shooter game.

When I started this project, I had no experience using the language (C#) or the framework (Unity). The idea was to teach myself something new while also exploring a passion of mine. Development was started in 2023 and finished in 2025, although this was intermittent and the project was not being actively worked on for most of 2024.

Technologies used:

  • C# and Unity
  • Aseprite for sprites/animations/background
  • Reaper for music and sound effects

RandoChoice (2021-2022)

RandoChoice is an Android app which will pick a random option from a list of choices.

I became interested in developing an Android app as I liked the idea of creating something that I could install and use on my phone. The recommended language for Android development is Kotlin which appealed to me as it was similar to other languages I knew at the time, so this project allowed me to explore a new language without a steep learning curve.

Features include:

  • Input a list of choices to be randomly selected from
  • Save, load, and delete lists from local storage

Technologies used:

  • Kotlin
  • Gradle
  • Android Studio
